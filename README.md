# access_control_system
*（实际实现效果比较简陋，参考价值低，仅为个人练习内容）*

## 📒概述
基于STM32F407+FreeRTOS+LVGL的智能门禁锁项目，能够通过密码、指纹、M1卡三种方式进行开门，同时通过显示屏选项可以对密码进行编辑，指纹、M1卡的信息进行添加、删除。

## 🚀
